If `form` is an instance of [`RunOrg.Form`](/forms/form.htm), then `form.Load()` fills (or replaces) its non-`id` fields with data retrieved from the API. Returns a [promise](/concepts/promise.htm) that will resolve to the `form` object if it could be filled. new RunOrg.Form("1ax69bfa335").Load().then(function(form) { console.log("Fetched data for %s.", form.label) }) var form = new RunOrg.Form("1ax69bfa335"); form.Load().then(function() { console.log("Fetched data for %s.", form.label) }, function(error) { if (error.HTTP == 404) console.log("Form %s not found.", form.id) else console.log("An error happened: %o", error) }) Note: it is not necessary to `Load()` a form to call its other methods. You can, for example, run `new RunOrg.Form(id).Stats()` to get the fill statistics for a form without loading the form itself. Returns a [`
`](/forms/form.htm) representation of the form with identifier `{id}` in database `{db}`.
